sitemapper-surveys
A specialized toolset for site mapping and survey data processing, optimized for high-performance CAD workstations.

🚀 Overview
sitemapper-surveys is designed to streamline the workflow between field survey data and 3D mapping environments. It focuses on high-speed data I/O and accurate spatial visualization.

✨ Key Features
Automated Data Processing: Rapid ingestion of raw survey coordinates.

CAD Integration: Optimized for 3D modeling and assembly workflows.

Large Dataset Handling: Engineered to leverage multi-drive SSD configurations for scratch disk performance.
